# INSTRUMENTO DE LIDERANÇA — IL
## O Pentagrama de Ginger — Módulo Diagnóstico
## 125 Questões — Perspectiva da Liderança

> **Instruções ao respondente:**
> Para cada afirmação abaixo, escolha o número que melhor representa sua percepção honesta sobre a realidade atual da empresa que você lidera ou co-lidera. Não responda como gostaria que fosse — responda como você acredita que é hoje.
> **1** = Discordo totalmente | **2** = Discordo parcialmente | **3** = Neutro | **4** = Concordo parcialmente | **5** = Concordo totalmente

## NOTA DE ESPELHAMENTO

Cada questão do IL corresponde diretamente à questão de mesmo número no IC. O espelhamento segue uma única lógica: a pergunta de referência para cada questão foi — "O que a liderança precisa acreditar sobre si mesma e sobre o campo que constrói para que essa realidade exista para os colaboradores?"

O IC captura o campo vivido. O IL captura o campo percebido pela liderança. A distância entre os dois é o Gap de Campo — a variável diagnóstica central do sistema.

Este instrumento precisa ser validado pelo autor Jovane Borlini da Silva antes de uso em aplicações formais.
